# 📐 市政排污管网工程量自动计量 APP

## 📋 项目信息

| 项目 | 内容 |
|------|------|
| **应用名称** | 管网计量助手 |
| **开发语言** | Python 3.10+ |
| **界面框架** | Streamlit (Web APP) |
| **核心功能** | CAD 提取、Excel 汇总、自动计量 |
| **开发时间** | 2026-03-22 |

---

## 🎯 功能模块

### 1️⃣ 数据导入模块
- 📐 DXF/CAD 文件读取
- 📊 Excel/CSV 数据导入
- 📝 手动输入界面
- 🗺️ 天地图坐标获取

### 2️⃣ 数据提取模块
- 🔍 管道信息提取（管径、材质、长度）
- 🕳️ 检查井信息提取（类型、深度）
- 📏 自动计算管道长度
- 📊 数据校验与纠错

### 3️⃣ 工程量计算模块
- 📐 管道长度统计
- 🕳️ 检查井数量统计
- 🚜 土方量计算（挖方/回填/外运）
- 🧱 附属设施计算

### 4️⃣ 报表输出模块
- 📊 Excel 工程量汇总表
- 📐 CAD 平面布置图
- 📄 PDF 计量报告
- 📱 移动端查看
