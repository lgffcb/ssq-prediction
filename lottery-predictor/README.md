# 🎱 双色球预测软件 v1.0

基于多种概率分布和深度学习的彩票预测分析工具

## ✨ 功能特性

### 📊 统计分析方法
- **伯努利分布** - 分析号码出现/不出现的概率
- **正态分布** - 分析和值的正态分布特性
- **指数分布** - 分析遗漏值的分布规律
- **二项分布** - 分析 n 期内号码出现次数
- **泊松分布** - 分析号码出现频率
- **LSTM 深度学习** - 基于时间序列的神经网络预测

### 🔍 分析维度
- 多周期分析（5 期、7 期、10 期、30 期、50 期）
- 冷热号分析
- 遗漏值分析
- 和值统计
- AC 指数分析
- 跨度分析

### 🎯 预测功能
- 综合多种方法生成预测号码
- 可配置权重（频率/遗漏/LSTM 等）
- 批量生成多组预测
- 支持自定义参数

## 🔬 技术原理

### 伯努利分布
分析每个号码在下一期出现（成功）或不出现（失败）的概率。

```python
P(X=1) = p  # 出现概率
P(X=0) = 1-p  # 不出现概率
```

### 正态分布
分析红球和值的分布特性，预测下期和值范围。

```python
和值 ~ N(μ, σ²)
95% 置信区间：[μ-1.96σ, μ+1.96σ]
```

### 泊松分布
分析号码出现频率的分布。

```python
P(X=k) = (λ^k * e^(-λ)) / k!
λ = 平均出现率
```

### LSTM 深度学习
使用长短期记忆网络分析时间序列模式。

```
输入：过去 N 期的开奖号码
LSTM 层 1: 128 单元
LSTM 层 2: 64 单元
LSTM 层 3: 32 单元
输出：下期预测号码
```
