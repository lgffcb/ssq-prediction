## 🎯 Proactive Agent 功能

Proactive Agent 是 OpenClaw 的内置功能，可以让 AI 主动：

1. **定时提醒** - 设置 cron 任务定时提醒
2. **定期检查** - Heartbeat 机制定期检查任务
3. **主动通知** - 发现重要信息时主动通知
4. **任务跟踪** - 跟踪未完成的任务并跟进

---

## 🛠️ 设置方法

### 方式 1：使用 Cron 定时任务

**设置提醒：**
```
提醒我下午 3 点开会
```

**定期检查：**
```
每 30 分钟检查一次邮件
```

### 方式 2：使用 Heartbeat

编辑 `HEARTBEAT.md` 文件，添加定期检查任务：

```markdown
# 定期检查任务
- 检查未读邮件
- 查看日历事件
- 检查项目进度
```

### 方式 3：创建主动代理脚本

我已经创建了主动代理脚本：

```bash
python3 tools/proactive_agent.py
```

## 🔧 技术实现

### Cron 任务结构
```json
{
  "name": "提醒任务",
  "schedule": {
    "kind": "at",
    "at": "2026-03-22T15:00:00+08:00"
  },
  "payload": {
    "kind": "systemEvent",
    "text": "提醒：下午 3 点开会"
  },
  "sessionTarget": "main"
}
```

### Heartbeat 检查
- 每隔 30-60 分钟自动唤醒
- 读取 `HEARTBEAT.md` 执行检查
- 发现重要事项时主动通知
