# 📊 数据自动提取汇总 Skill

## 📋 技能信息

| 项目 | 内容 |
|------|------|
| **技能名称** | DataExtractor Pro |
| **开发时间** | 2026-03-22 |
| **应用场景** | 自动提取数据并汇总 |
| **支持格式** | Excel/CSV/DXF/JSON/TXT |
| **熟练度** | ⭐⭐⭐⭐ 可用 |

---

## 🎯 核心功能

### 1️⃣ 多格式数据导入
- 📊 Excel (.xlsx, .xls)
- 📄 CSV (.csv)
- 📐 DXF (.dxf) - CAD 文件
- 📝 JSON (.json)
- 📋 TXT (.txt)

### 2️⃣ 智能数据提取
- 🔍 自动识别列名
- 📏 自动计算数值
- 🔢 自动分类统计
- ⚠️ 异常值检测

### 3️⃣ 数据汇总分析
- 📊 分类汇总
- 📈 趋势分析
- 📉 对比分析
- 🎯 关键指标

### 4️⃣ 报表输出
- 📄 Excel 汇总表
- 📊 图表可视化
- 📝 Markdown 报告
- 📐 PDF 报告

## 📊 使用示例

### 示例 1：Excel 数据汇总

**输入：** 多个 Excel 文件
```
data/
├── 2026-01.xlsx
├── 2026-02.xlsx
└── 2026-03.xlsx
```

**输出：** 汇总 Excel
```
output/
└── 季度汇总.xlsx
```

### 示例 2：CAD 数据提取

**输入：** 管网 DXF 文件
```
data/
└── 管网平面图.dxf
```

**输出：** 工程量表
```
output/
├── 管道工程量.xlsx
└── 检查井工程量.xlsx
```
