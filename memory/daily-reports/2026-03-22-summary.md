### 1️⃣ 抖音视频文案提取

**工具：** AnyToCopy (https://anytocopy.com/)

**操作流程：**
```
1. 打开 AnyToCopy 网站
2. 粘贴抖音视频链接
3. 点击"开始提取"
4. 等待 AI 语音识别完成
5. 复制文案内容
```

**关键发现：**
- 需要抖音登录才能获取完整字幕
- 短链接需要先跳转到完整链接
- 文案识别准确率约 95%

**输出物：**
- 完整文案 Excel 文件
- 音频 MP3 下载链接
- 视频无水印下载

### 3️⃣ 天地图卫星图查看

**工具：** 天地图 (https://map.tianditu.gov.cn/)

**URL 格式：**
```
https://map.tianditu.gov.cn/?c=经度，纬度&l=缩放级别&v=地图类型
```

**参数说明：**
- `c=经度，纬度` - 地图中心点
- `l=1-20` - 缩放级别（20 最清晰）
- `v=1` - 矢量图，`v=2` - 影像图（卫星图）

**示例：**
```
https://map.tianditu.gov.cn/?c=115.219444,23.835362&l=19&v=2
```

**应用场景：**
- 🏗️ 施工地点定位
- 📏 距离测量
- 🗺️ 范围查看
- 📸 截图保存

**API Key：** 55362d559694da0d42206960bd48e98c（浏览器端）
