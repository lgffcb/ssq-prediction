### 2026-03-22 技能学习记录

#### 🗺️ 天地图卫星图切换技能

**学会的操作流程：**

```
1. 搜索地点 → 输入"黄村坳村"
2. 点击搜索按钮 → 获取搜索结果
3. 切换到影像图 → 使用 URL 参数 `v=2`
4. 设置缩放级别 → `l=19` 或 `l=20`
5. 设置中心点 → `c=经度，纬度`
6. 截图保存 → 获取卫星图
```

**正确的 URL 格式：**
```
https://map.tianditu.gov.cn/?c=115.219444,23.835362&l=19&v=2
```

**参数说明：**
- `c=经度，纬度` - 地图中心点坐标
- `l=缩放级别` - 1-20，20 为最高清
- `v=地图类型` - 1=矢量图，2=影像图（卫星图）

**关键发现：**
- 天地图 API Key 分类型（浏览器端/服务器端）
- 浏览器端 Key 只能在网页中使用，不能直接调用 API
- 切换卫星图最直接的方式是修改 URL 参数 `v=2`

**下次操作优化：**
1. 直接用 URL 参数切换，不点击 UI 按钮
2. 先用 snapshot 获取最新元素 ref
3. 操作后立即 screenshot 确认结果
