## ⚠️ 注意事项

1. **首次启动** 需要安装依赖
   ```bash
   pip install streamlit ezdxf pandas openpyxl
   ```

2. **端口占用** 如 8501 端口被占用
   ```bash
   streamlit run app/main.py --server.port 8502
   ```

3. **文件权限** 确保 output 目录可写
   ```bash
   mkdir -p output && chmod 755 output
   ```

4. **浏览器访问** 支持 Chrome/Firefox/Edge

## 💡 使用技巧

### 快速估算
- 户均管长按 15-25m 估算
- 检查井间距按 30-50m 设置
- 埋深按 1.2-2.0m 设置

### CAD 提取
- 确保 DXF 文件是 R2010 格式
- 管道用多段线绘制
- 检查井用圆表示

### 手动输入
- 坐标格式：x,y（如 100,200）
- 长度自动计算
- 实时保存数据
