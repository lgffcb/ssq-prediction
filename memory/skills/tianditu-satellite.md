## 🎯 标准操作流程

### 方式一：URL 参数直接切换（推荐）

```
https://map.tianditu.gov.cn/?c=经度，纬度&l=缩放级别&v=地图类型
```

**参数说明：**
- `c=经度，纬度` - 地图中心点（如：115.219444,23.835362）
- `l=1-20` - 缩放级别（20 为最高清）
- `v=1 或 2` - 1=矢量图，2=影像图（卫星图）

**示例：**
```
https://map.tianditu.gov.cn/?c=115.219444,23.835362&l=19&v=2
```

### 方式二：浏览器操作

```
1. 打开天地图官网
2. 搜索地点名称
3. 点击右侧工具栏的"影像"按钮
4. 使用 +/- 按钮调整缩放级别
5. 截图保存
```

## ⚠️ 注意事项

1. **API Key 类型**
   - 浏览器端：只能网页使用
   - 服务器端：可以 API 调用
   - 根据需求选择

2. **坐标格式**
   - 经度在前，纬度在后
   - 用逗号分隔，无空格
   - 小数点后保留 6 位

3. **缩放限制**
   - 最高 20 级
   - 乡村地区可能只有 18-19 级清晰
   - 城市地区可达 20 级

4. **截图时机**
   - 等地图完全加载后再截图
   - 使用 timeoutMs=10000 确保加载完成

## 💡 常见问题

### Q1: 切换卫星图没反应？
**A:** 检查 URL 参数 `v=2` 是否正确，或刷新页面重试

### Q2: 地图不清晰？
**A:** 继续放大到 19-20 级，或检查网络加载

### Q3: API 调用失败？
**A:** 浏览器端 Key 不能用于 API 调用，改用网页方式
